# Project-Defiance
For the UPCSG Game Jam 2026 by Vince Antolihao, Russell Beduya, Angelo Manlangit, CJ Retiza, and Cole Unson <br>
Project-Defiance is a 3D Platformer where the theme is defiance (path less taken in society) where you gotta complete a level fast and find shortcuts wherever you can. It is built around speed, creativity, and rebellion. Instead of following preset routes, players are encouraged to experiment, break conventions, and find unconventional shortcuts. Every level rewards curiosity and bold movement because the fastest path is rarely the obvious one.

**Gameplay** <br>
Controls: <br>
WASD - Movement <br>
Space - Jump <br>
Left Click - Use equipped item/punch <br>
Move around the level to reach the final goal. <br>
Some items are scattered, can pick them up by going near them. <br>
Level usually has hidden shortcuts so use your creativity to find them or make your own. <br>
When a new item is picked up, old item gets discarded.

**How To Use** <br>
Open the Project: Clone the repository and open project.godot using Godot Engine 4.5. <br>
Run the Game: Press F5 or the Play button in the top right corner of the editor to start the game.

**Build Instructions (itch.io)** <br>
This game is designed to be submitted as a Web or Desktop build for game jam submission: <br>
Go to Project > Export. <br>
Add a Web preset. <br>
Ensure the "VRAM Compression" is compatible with web browsers. <br>
Export the project into a dedicated folder, zip the contents (ensuring index.html is at the root), and upload to itch.io.
